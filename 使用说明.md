# 📱 快速使用指南

## 🚀 立即开始

### 方法一：本地预览
```bash
# 使用Python本地服务器
python -m http.server 8000

# 或使用Node.js
npx http-server

# 访问地址
http://localhost:8000/index_mobile_enhanced.html
```

### 方法二：GitHub Pages部署
```bash
# 1. 上传文件到GitHub仓库
git add .
git commit -m "feat: 移动端增强版"
git push

# 2. 在仓库设置中启用Pages
# Settings → Pages → Source: Deploy from a branch → main

# 3. 访问你的网站
# https://你的用户名.github.io/仓库名/index_mobile_enhanced.html
```

## 📋 核心功能使用

### 🔍 搜索功能
**位置**：页面顶部搜索框
**操作**：
1. 点击搜索框
2. 输入品牌名/商品关键词（如："Hermes"、"包包"）
3. 实时显示搜索结果

---

### 🎛️ 筛选功能
**位置**：搜索框下方
**操作**：
1. **价格筛选**：选择价格区间
2. **品牌筛选**：输入品牌名称（支持模糊匹配）
3. **清除筛选**：点击按钮重置所有条件

---

### 📤 分享功能
**位置**：每张图片左上角（粉色圆形按钮）
**操作**：
1. 点击粉色分享按钮
2. 选择分享方式：
   - 💬 **微信好友**：复制分享内容到剪贴板
   - 📱 **微博**：打开微博分享页面
   - 🐧 **QQ**：打开QQ分享页面
   - 🔗 **复制链接**：复制页面链接

---

### 🛒 购买同款
**位置**：每个穿搭卡片的商品列表
**操作**：
1. 点击任意商品项
2. 自动复制商品信息到剪贴板
3. 显示复制成功提示
4. 1.5秒后自动跳转到淘宝

---

### 📖 浏览集数
**位置**：页面顶部导航栏
**操作**：
1. 点击集数按钮（如"第1集"）
2. 查看该集的所有穿搭
3. 点击"展开更多集数"查看后续集数

---

### 🖼️ 查看大图
**位置**：穿搭卡片的主图
**操作**：
1. 点击图片
2. 弹出高清大图预览
3. 点击空白处或按ESC关闭

---

### ⬆️ 返回顶部
**位置**：页面右下角（粉色圆形按钮）
**操作**：
1. 滚动页面时自动显示
2. 点击按钮快速回到顶部

## 🔧 自定义配置

### 修改主题色
编辑`style_v2.css`文件中的CSS变量：
```css
:root {
    --primary-color: #ff9ec4;      /* 主色调：粉色 */
    --secondary-color: #b794f6;    /* 辅助色：紫色 */
    --accent-color: #fbcfe8;       /* 强调色：浅粉 */
}
```

### 添加更多分享方式
在`script_enhanced.js`的`initShareMenu()`函数中添加：
```javascript
<div class="share-option" onclick="shareToXXX()">
    <div class="share-option-icon" style="background: #颜色码;">图标</div>
    <div class="share-option-label">平台名</div>
</div>
```

### 修改分享文案
编辑`script_enhanced.js`中的分享函数：
```javascript
const shareText = `你的自定义分享文案\n${window.location.href}`;
```

## ⚠️ 注意事项

### 浏览器兼容性
- ✅ **Chrome** 66+
- ✅ **Firefox** 63+
- ✅ **Safari** 13.1+
- ✅ **微信内置浏览器**（部分功能）
- ⚠️ **Internet Explorer** 不支持

### 移动端分享限制
- **微信**：需要手动复制分享内容到微信
- **微博/QQ**：会自动打开对应分享页面
- **HTTPS**：分享功能需要HTTPS环境或localhost

### 网络要求
- **首屏加载**：约1-2秒（取决于网络）
- **图片加载**：支持懒加载（待实现）
- **离线访问**：不支持（待实现PWA）

## 📊 性能优化建议

### 1. 图片压缩
当前图片总大小：44MB
建议：压缩至20MB以内
```bash
# 使用imagemagick压缩
mogrify -quality 80 -resize 800x pic/*/*.jpg
```

### 2. 启用CDN
将图片迁移至CDN（如七牛云、阿里云OSS）
更新`outfits.json`中的图片路径

### 3. 图片格式转换
将JPG转换为WebP格式，可减少50%体积
```bash
# 使用cwebp转换
cwebp -q 80 input.jpg -o output.webp
```

## 🎯 常见问题

### Q: 页面加载慢怎么办？
A: 检查网络连接，或启用图片CDN加速

### Q: 分享到微信没反应？
A: 微信内浏览器限制，已自动降级为复制分享内容

### Q: 搜索不准确？
A: 尝试使用更精确的关键词或品牌名

### Q: 图片显示不全？
A: 刷新页面或检查图片路径是否正确

### Q: 移动端体验不佳？
A: 确保使用现代浏览器（Chrome、Safari等）
